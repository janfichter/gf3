Плагин для создания и отображения интерактивных семейных деревьев на сайте WordPress с поддержкой GEDCOM, корневых элементов и расширенной навигацией.

== Description ==

Плагин «Genius Family Tree» — это мощный инструмент для создания и визуализации семейных древ. Версия 1.3.4 включает множество улучшений и новых возможностей.

🚀 **Ключевые возможности:**

* **Интерактивное древо** — создавайте красивые, масштабируемые семейные деревья с помощью библиотеки D3.js.
* **Корневой элемент древа** — вы можете указать, от какого члена семьи должно строиться основное древо (звездочка в списке персоналий).
* **Переход от конкретной персоны** — на странице каждой персоны есть кнопка «Показать в древе», которая строит древо именно от этого человека (URL параметры `person_id` или `center_on`).
* **GEDCOM импорт/экспорт** — полная поддержка формата GEDCOM 5.5.1 с сохранением фотографий и корневого элемента.
* **Каталог фамилий** — удобная навигация по фамилиям с алфавитным указателем (шорткод `[family_surname_catalog]`).
* **Расширенная админка** — столбцы с портретами, годами жизни и индикацией корневого элемента.

🎯 **Новые функции в версии 1.3.4:**

* **Умный выбор корневого элемента**: 
  - Можно установить глобальный корень древа в настройках
  - При переходе со страницы персоны автоматически строится древо от неё
  - Приоритет: URL параметр → глобальный корень → data-атрибут
* **Поддержка двух URL параметров**: `person_id` и `center_on` для обратной совместимости
* **Визуальная индикация**: в списке персоналий отображается звездочка у корневого элемента
* **Быстрое редактирование**: можно установить корневой элемент прямо из списка персон

== Installation ==

= СПОСОБ 1: УСТАНОВКА ЧЕРЕЗ АДМИНКУ WORDPRESS =

1. Скачайте плагин (архив .zip).
2. Войдите в админку WordPress (ваш-сайт.ru/wp-admin).
3. Перейдите в «Плагины» → «Добавить новый».
4. Нажмите кнопку «Загрузить плагин» (в верхней части страницы).
5. Выберите скачанный архив .zip и нажмите «Установить».
6. После установки нажмите «Активировать плагин».

= СПОСОБ 2: РУЧНАЯ УСТАНОВКА ЧЕРЕЗ FTP =

1. Распакуйте архив с плагином.
2. Подключитесь к серверу через FTP.
3. Перейдите в папку `/wp-content/plugins/`.
4. Загрузите папку `genius-family-tree` на сервер.
5. Перейдите в админку WordPress → «Плагины».
6. Найдите «Genius Family Tree» в списке и нажмите «Активировать».

== Configuration ==

= Основные настройки =

1. После активации плагина в меню появится новый пункт «Семейное древо».
2. Нажмите «Семейное древо» → «Настройки» для доступа к настройкам плагина.
3. Здесь вы можете ввести лицензионный ключ и управлять GEDCOM функциями.

= Установка корневого элемента =

**Способ 1 (автоматический при импорте):**
1. При экспорте GEDCOM корневой элемент автоматически помечается тегом `_ROOT Y`
2. При импорте этого файла корневой элемент восстанавливается автоматически

**Способ 2 (ручной):**
1. Перейдите в «Семейное древо» → «Все члены семьи»
2. Найдите нужную персону и нажмите «Редактировать»
3. В правой колонке найдите метабокс «Корневой элемент древа»
4. Отметьте галочку «Сделать корневым элементом древа»
5. Сохраните запись

**Способ 3 (быстрое редактирование):**
1. В списке персоналий наведите на нужную запись
2. Нажмите «Быстрое редактирование»
3. Отметьте галочку «Сделать корневым элементом древа»
4. Нажмите «Обновить»

= Шорткоды =

**Основное древо:**
[family_tree]


Отображает интерактивное древо от глобального корневого элемента.

**Древо от конкретной персоны:**
[family_tree root="123"]


Строит древо от персоны с ID 123.

**С фокусом на персоне:**
[family_tree root="123" center_on="456"]


Строит древо от персоны 123, но центрирует на персоне 456.

**Каталог фамилий:**
[family_surname_catalog]


Отображает алфавитный каталог всех фамилий с ссылками на персоналии.

== Frequently Asked Questions ==

= Как работает система корневого элемента? =

Плагин использует умную систему приоритетов:
1. **Наивысший приоритет**: URL параметры (`?person_id=123` или `?center_on=123`) — при переходе со страницы персоны
2. **Средний приоритет**: Глобальный корень из настроек — значение по умолчанию для основного древа
3. **Низший приоритет**: Data-атрибут `data-root-id` — для обратной совместимости

= Как перейти к древу от конкретной персоны? =

**Способ 1:** На странице персоны нажмите кнопку «Показать в древе».
**Способ 2:** Добавьте к URL страницы с древом параметр `?person_id=123` (где 123 — ID персоны).
**Способ 3:** Используйте параметр `?center_on=123` (для старых ссылок).

= Как узнать ID персоны? =

ID персоны отображается в админ-панели в списке всех членов семьи (колонка «ID») или в URL при редактировании персоны (например, `post=123`).

= Какие данные сохраняются в GEDCOM? =

При экспорте сохраняются:
- Все персональные данные (имя, фамилия, даты, места)
- Родственные связи (родители, дети, супруги)
- **Корневой элемент** (тег `_ROOT Y`)
- Фотографии (URL и локальный путь)
- Примечания

= Есть ли рекомендации по заполнению? =

* Начинайте с самого старшего поколения — это поможет правильно построить связи.
* Заполняйте обязательные поля — имя, фамилия, пол.
* Используйте поиск при добавлении связей — в полях отец/мать/супруги начните вводить имя, и система предложит варианты.
* Устанавливайте корневой элемент — это определит, от кого будет строиться основное древо.

= Что делать если древо не отображается? =

1. Проверьте, добавлены ли члены семьи.
2. Убедитесь, что шорткод `[family_tree]` добавлен на страницу.
3. Проверьте, установлен ли корневой элемент (должна быть звездочка в списке персоналий).
4. Откройте консоль браузера (F12 → Console) и посмотрите, есть ли ошибки.
5. Убедитесь, что скрипты не заблокированы (например, AdBlock).

= Что делать если корневой элемент не сохраняется при импорте? =

Убедитесь, что:
1. В экспортированном GEDCOM файле есть тег `_ROOT Y` у нужной персоны.
2. Вы используете Pro-версию плагина (функция доступна только в платной версии).
3. При импорте нет ошибок в логах.

= Что делать если переход по ссылке `?center_on=123` не работает? =

Проверьте:
1. Существует ли персона с таким ID.
2. Есть ли у этой персоны связи (родители/дети/супруги).
3. Откройте консоль браузера — должно быть сообщение `🎯 Используем ID из URL (center_on): 123`

= Что делать если возникают ошибки при добавлении персон? =

1. Проверьте, не превышен ли лимит в 50 персоналий (в бесплатной версии).
2. Убедитесь, что обязательные поля заполнены (имя, фамилия, пол).
3. Проверьте логи ошибок WordPress (файл `/wp-content/debug.log`).

= Что делать если возникли проблемы с плагином? =
